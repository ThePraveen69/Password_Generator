# 🔐 Python Password Generator

A simple and customizable command-line tool built with Python to generate secure, random passwords based on user-defined criteria.

## 🚀 Features
- **Adjustable Length:** Generate passwords of any minimum length.
- **Customizable Complexity:** Choose whether to include:
  - Uppercase and Lowercase letters (Standard)
  - Numbers (Optional)
  - Special Characters/Symbols (Optional)
- **Randomized Logic:** Uses Python's `random` and `string` modules for unpredictable outputs.

## 🛠️ Requirements
- **Python 3.x** installed on your machine.
- No external libraries required (uses built-in modules).
